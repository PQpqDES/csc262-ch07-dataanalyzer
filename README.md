CharacterCounter

This program asks the user to type in a file name.

After the user enters the file name, the program tries to open the file. 
If the file is found, it counts the number of characters, words, and lines in the file.

The program has three methods:
countCharacters
countWords
countLines

Each method opens the file and counts one thing.

The program uses try-catch so it does not crash if the file is not found.
If the file is not found, it prints "File not found."

This was written for a CSC262 assignment.
